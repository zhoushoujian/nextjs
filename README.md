# nextjs-example

`nextjs服务端渲染的空壳项目,集成常用的框架,如redux和less,方便快速初始化一个项目`

## Functions

nextjs + redux + typescript + express + less

### Install

```bash
npm install
npm run start
# or
yarn
yarn start
```
