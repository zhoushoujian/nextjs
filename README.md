# nextjs-example

`nextjs服务端渲染的空壳项目,集成常用的框架,如redux和less,方便快速初始化一个项目`

## Functions

nextjs + redux + typescript + express + less + css module

## Install

```bash
npm install
npm run start
# or
yarn
yarn start
```

## Details

1. 服务端和客户端都接入@来导入模块

2. 接入指定依赖检查

3. 处理跨域

4. 处理静态资源

5. 拆分页面路由和 api 网关接口

6. 封装页面请求响应和 api 请求响应

7. 接入日志系统

8. 封装请求 hooks

9. 接入 redux-logger 并挂在 store 状态到全局

10. 为每一次请求分配 id
