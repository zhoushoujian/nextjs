# shuyun-ssr-cli

## 安装

```shell
npm i shuyun-ssr-cli@latest -g
```

## 使用

`初始化项目`

```shell
shuyun-ssr-cli init
# 或
shuyun-ssr-cli i
```

`检查版本`

```shell
shuyun-ssr-cli -V
# 或
shuyun-ssr-cli --version
```

`显示帮助`

```shell
shuyun-ssr-cli -h
# 或
shuyun-ssr-cli --help
```

## 调试

```shell
npm run debug
```
